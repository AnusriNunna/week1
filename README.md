Personal Portfolio Website
Project Overview

This project is a Personal Portfolio Website created using HTML5 and basic CSS. The website showcases personal information, technical skills, and contact details in a structured and professional format. It is designed to demonstrate an understanding of HTML fundamentals, semantic elements, and proper project organization suitable for academic and beginner-level professional use.

Objectives

To learn and implement HTML5 document structure

To use semantic HTML elements for better readability and accessibility

To create multiple sections such as About, Skills, and Contact

To implement internal navigation within a single webpage

To build a contact form with basic input validation

To organize files properly for GitHub submission

Technologies Used

HTML5

CSS (internal styling)

Setup and Installation Instructions

Install a code editor (Visual Studio Code recommended)

Create a project folder named portfolio-website

Add the following files and folders:

index.html

README.md

images folder

Place a profile image inside the images folder

Open index.html in any modern web browser

No external libraries or installations are required.
